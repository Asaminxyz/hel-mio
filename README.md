# HÉLMIO website

## 編集するファイル

- `talents.js`
  - 写真
  - YouTube
  - ボイスサンプル
  - プロフィール
  - 実績

## 写真の置き場所

`images/` に入れてください。

例：

- `images/asami.jpg`
- `images/rina.jpg`

## 音声の置き場所

`audio/` に入れてください。

例：

- `audio/asami.mp3`

## YouTubeの入れ方

通常URLではなく、埋め込みURLを使います。

例：

通常：
`https://www.youtube.com/watch?v=XXXXXXXXXXX`

埋め込み：
`https://www.youtube.com/embed/XXXXXXXXXXX`

## フォーム設定

`index.html` の以下を Formspree のURLに差し替えてください。

- `YOUR_CONTACT_FORM_ID`
- `YOUR_AUDITION_FORM_ID`
